## **AURA OS** 

Один запрос — целая команда ИИ. Операционная среда, в которой агенты работают вместе. 

Маркетинговое предложение 

Версия 1.0.0 · Июнь 2026 

## **Проблема** 

У вас уже есть сильные ИИ-инструменты: Claude Code, Codex, Gemini, локальные модели. Но каждый живёт в своём окне, ничего не знает о других и забывает контекст после каждой сессии. Человек вручную носит результаты из одного чата в другой, сам проверяет ошибки и сам помнит, «что мы уже делали». Это медленно и не масштабируется. 

## **Решение — AURA OS** 

AURA OS объединяет ваших агентов в слаженную команду с разделением ролей, параллельной работой, взаимной проверкой и общей памятью. Вы ставите задачу — голосом или текстом. Дальше система всё делает сама: планирует, распределяет, исполняет, проверяет, исправляет и запоминает. 

|**Возможность**|**Что это даёт**|
|---|---|
|Умная декомпозиция задач|Координатор разбивает задачу на подзадачи и подбирает исполните|
|Параллельная работа|Независимые подзадачи выполняются одновременно — большие зад|
|Перекрёстное ревью|Один агент пишет — другой проверяет. Цикл «ошибка → исправлени|
|Общая память в Obsidian|Каждая задача становится заметкой в вашем vault. Команда агентов|
|Любые агенты, включая лок|альные<br>Claude Code, Codex, Gemini, Hermes, Ollama — и любой ваш CLI-инстр|
|Голос и текст|Диктуйте задачи голосом или печатайте — как удобно.|
|Конфиденциальность|Никаких новых ключей и облаков: агенты используют свои собственн|



## **Как это выглядит** 

«Создай приложение-таймер на Python, проверь и исправь ошибки» → Claude Code пишет код → Codex параллельно готовит тесты → Codex проводит ревью и находит 3 проблемы → Claude Code исправляет → повторное ревью: APPROVED → готовое приложение в рабочей папке, отчёт — в Obsidian. Ваше участие: одна фраза. 

## **Для кого** 

Разработчики и команды — конвейер «код + ревью + тесты» без ручной рутины. Исследователи и аналитики — параллельный сбор и сведение информации с накоплением базы знаний. Создатели контента — текст, проверка фактов и упаковка материалов в одном потоке. Энтузиасты локальных ИИ — приватный мультиагентный стек на Ollama без облака. 

## **Почему AURA OS, а не ещё один чат** 

||**Обычный ИИ-чат**|**AURA OS**|

|Исполнителей|один|команда агентов + субагенты|



||**Обычный ИИ-чат**|**AURA OS**|

## **Предложение** 

AURA OS 1.0 распространяется бесплатно: исполняемый файл для Windows и полный исходный код. Установите, подключите своих агентов и поставьте первую задачу — через пять минут у вас будет работающая команда ИИ.
